# Frontend

This project is created by using Vite and Vue3.

To start with Vite go here https://vitejs.dev/

## Create project

To create a project simply use:
    
```bash
yarn create vite
# OR
bunx create-vite
```

## Install dependencies

To install dependencies use:

```bash
cd frontend
yarn install
# OR
bun i
```

## Run project

To run project use:

```bash
yarn dev
# OR
bun dev
```
